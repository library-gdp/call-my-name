# 자기소개서 작성 가이드

## 문항별 초점

- 지원동기: 회사의 구체적인 사업·제품·조직 방향과 지원자의 경험이 만나는 지점을 설명한다.
- 직무역량: JD의 핵심 과업 하나를 선택하고, 유사한 문제를 해결한 행동과 결과로 입증한다.
- 성장과정: 연대기 대신 현재의 업무 태도를 만든 사건과 이후 반복된 행동을 보여준다.
- 협업·갈등: 상대의 관점, 본인의 구체적 조정 행동, 공동 결과, 배운 점을 포함한다.
- 실패·도전: 실패를 미화하지 말고 원인 분석, 수정 행동, 재발 방지 또는 후속 결과를 제시한다.
- 입사 후 포부: 단기 학습, 중기 기여, 장기 성장 순으로 쓰되 회사가 약속하지 않은 직무나 사업을 단정하지 않는다.

## 근거 강도

강한 근거부터 우선 사용한다.

1. 수치와 산출물로 확인되는 직접 경험
2. 구체적인 역할·행동·결과가 있는 프로젝트
3. 관련 교육, 자격, 포트폴리오
4. 관심이나 의지만 있는 주장

## 퇴고 체크리스트

- 첫 문단에서 문항에 답했는가?
- 본인의 역할과 팀의 성과를 구분했는가?
- 행동을 추상어 대신 동사로 표현했는가?
- 수치의 기준과 기간이 명확한가?
- 기업명만 바꾸면 재사용되는 상투적 문장을 삭제했는가?
- 확인되지 않은 회사 전략, 문화, 성과를 단정하지 않았는가?
- 과도한 감탄, 미사여구, 같은 접속어 반복을 줄였는가?
